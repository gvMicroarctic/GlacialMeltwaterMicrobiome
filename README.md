# GlacialMeltwaterMicrobiome

Welcome to the GlacialMeltwaterMicrobiome GitHub repository. This repository hosts a collection of bioinformatic and plotting scripts used in the research paper titled *“Microorganisms and Their Mobile Genetic Elements in Glacial Meltwater: A Dynamic System for Gene Exchange.”*

The bioinformatic analysis pipeline can be found [here](https://html-preview.github.io/?url=https://github.com/gvMicroarctic/GlacialMeltwaterMicrobiome/blob/main/icevirome_bioinformatics.html). Associated scripts are available in the “BIOINFORMATICS” subfolder.

Scripts used to generate the R plots presented in the manuscript are located in the “PLOTS” subfolder. Additional information about these scripts can be found [here](https://html-preview.github.io/?url=https://github.com/gvMicroarctic/GlacialMeltwaterMicrobiome/blob/main/icevirome_plots.html).
